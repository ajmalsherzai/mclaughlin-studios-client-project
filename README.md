# McLaughlin Studios Client Project

## Project Overview
This project summarizes a client-based MIS project completed for McLaughlin Studios. The project involved website planning, database documentation, business analysis, Tableau visuals, and project management deliverables.

The goal of this project was to support a small business client by organizing requirements, planning system features, documenting database needs, and creating business-focused technology recommendations.
