# McLaughlin Studios Client Project Summary

## Project Purpose
The purpose of this project was to support McLaughlin Studios through a client-based MIS project focused on website planning, database documentation, business analysis, and technology recommendations.

This project connected classroom MIS concepts with a real small business client.

## Project Goals
- Understand the client’s business and website needs
- Document business and system requirements
- Plan website features and improvements
- Organize database and data-related needs
- Create business-focused technology recommendations
- Support project planning and deliverables

## Work Completed
- Reviewed client business needs
- Organized website feature requirements
- Documented system and business requirements
- Planned database structure and data needs
- Created project documentation
- Developed Tableau/business visualization concepts
- Supported project timeline and deliverable planning
